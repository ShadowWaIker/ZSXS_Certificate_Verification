# 掌上忻师 - 证书生成 #

该系统可与证书核验功能(掌上忻师)配套使用，包含以下内容：

1. 一键生成当日导入系统的各类证书、文书或通知 *[PDF文档]*
2. 一键生成系统中的全部证书、文书或通知 *[PDF文档]*
3. 生成的证书可使用掌上忻师中的证书核验功能查验真伪
